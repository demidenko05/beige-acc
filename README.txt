Beigesoft™ Enterprise Information System Java code.
It consists of double entry accounting and trading (web-store) business logic.

To compile you need PostgreSql and MySql databases bseist with user/password beigeaccounting, see the test package for details.

To build run:
mvn clean install

To build offline docs run:
mvn site:site

* if you build the site from the maven's source 1.1 archive, then unpack and copy site's folders "site/gimp", "site/images" and "site/files" from the previous EIS1.0 source.
