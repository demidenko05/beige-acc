Беижсофт™ ПО для быстрой разработки кросс-платформенного финансового ПО с ВЕБ-интерфейсом.

* Кросс-платформенное означает что все части приложения работают одновременно на любой операционной системе. Например локальная (Интернет-недоступная) версия для Андроида включает в себя и сервер базы данных и сервер приложения (ВЕБ сервер), ВЕБ-клиент - Браузер-Хром работает здесь же.

Данная документация предназначена для широкой аудитории. Например материал рассчитан и для программистов кто ранее не использовал Джаву, и для тех кто хочет оценить трудоемкость программирования (изменения, дополнения) Беижсофт ИСП.

В начале будет использоваться легкая, не требующая установки СКЛайт (SQLite) база данных. В конце мы скопируем все данные из локальной СКЛайт версии в облачную, работающую на Постгрес (Postgres) или МайЭсКуЭл (MySQL). Также будет приведена информация по защите данных, например проверка цифровой подписи файла.
